## Pré-requisitos de Projeto
Todos os Projetos de Tempo Livre possuem pré-requisitos que você deve cumprir antes de poder realizá-los.

### Item Pré-requisito
Muitos Projetos de Tempo Livre possuem um ou mais itens especiais que você deve possuir ou obter antes que possa iniciar o projeto. Por exemplo, você não pode construir um dirigível sem primeiro encontrar Cristal de Vento de Quintessência, um componente raro e crítico na construção desse veículo. Outros projetos possuem um pré-requisito que envolve realizar certas atividades durante o processo do projeto.

### Origem do Projeto
Para iniciar um Projeto de Tempo Livre, você deve ter acesso a conhecimento específico com detalhes de como o projeto deve ser realizado. Isto pode estar na forma de informação escrita (um livro, pergaminho, esquema, etc.), um tutor especialista com sabedoria para compartilhar, um mestre artesão com habilidades para ensinar, uma manifestação sobrenatural do escopo e segredos do projeto ou qualquer outra forma determinada pelo Diretor. Este conhecimento focado no projeto é chamado Origem do Projeto. Você não só precisa de uma Origem do Projeto para iniciar um Projeto de Tempo Livre, como também precisa de acesso a essa Origem do Projeto sempre que realizar um Lançamento de Projeto para este projeto em particular (veja Lançamento de Projeto a seguir).

Algumas Origens do Projeto são relativamente fáceis de encontrar. Construir um castelo não requer tomos de conhecimento antigo, mas também não é algo que qualquer pedreiro sabe fazer. Entretanto, a localização da Tumba de Ashrya, a múmia protetora de uma centena de feiticeiros-soberanos adormecidos, pode existir em apenas um único tomo que requer uma aventura para descobrir.

O idioma com o qual uma Origem do Projeto está escrita ou pode ser compartilhada é definido pelo Diretor. Os projetos de exemplo a seguir possuem sugestões com idiomas de Orden (veja essa seção no Capítulo 4: Antecedentes), embora você possa encontrar uma versão de Origem do Projeto em outros idiomas também.

Sempre que o Diretor determinar que uma criatura com o conhecimento necessário pode ser usada como Origem do Projeto, outro herói pode servir como esta Origem do Projeto. Porém, esse herói precisa usar uma atividade de Descanso para ser a Origem do Projeto sempre que você realizar um Lançamento de Projeto para este projeto.

#### Procurar por Origens de Projeto Comuns
Se você precisar de uma Origem do Projeto para um projeto em particular, pergunte ao Diretor se a Origem do Projeto é comum o bastante que possa ser encontrada em uma libraria grande o suficiente, entre um grupo de sábios ou artesãos, etc. Se o Diretor concordar, então na próxima vez que realizar um Descanso em um local com uma biblioteca ou acesso a criaturas com o conhecimento desejado, você pode localizar a Origem do Projeto que precisa como uma atividade de Descanso.
